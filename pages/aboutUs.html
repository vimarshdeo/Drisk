So, Who are we?

    This game is developed by the graduate students of Arizona State University as a part of their fall semester project. This game is a web based client server version of the board game Risk.

The Team!!

    The team consists of five members who actively worked on different sections for developing this website and the game. Some of them handled the UI part while other worked on the server part.

    It took us around 2 months to complete this project develpoing the core features os the game.
